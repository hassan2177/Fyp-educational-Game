# Fyp-educational-Game
Educational game for fyp
The educational game is made in unity 3d Engine,Following  Assets are used:


UNITY ASSET STORE

Game Environment;
Low Poly Houses Free Pack
Publisher:Palmov Island
https://assetstore.unity.com/packages/3d/props/exterior/low-poly-houses-free-pack-243926


Car Controller;
PROMETEO: Car Controller
Publisher:Mena
https://assetstore.unity.com/packages/tools/physics/prometeo-car-controller-209444#publisher

Some custom UI Elements From figma  were also added by me.

GameObject collectable  Animations also made 

InOrder to run the project:
download the zip from this repo
extract and open in unity Engine 
after opening the scene Download Required Packages such as textmeshpro from Package manager window.
